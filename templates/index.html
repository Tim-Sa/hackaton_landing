Turtle Heroes

Добро пожаловать на хакатон!

Хакатон длится 60 минут, постарайся за это время набрать как можно больше баллов.

Ты можешь сконцентрироваться на одном из шести нижеперечисленных заданий или попробовать выполнить несколько из них.

Также количество баллов можно увеличить:

if названия переменных и функций понятны, осмысленны и написаны на английском языке:
    к награде за задание прибавляется +1 Балл

if код решения организован в компактные функции:
    к награде за задание прибавляется +1 Балл

if хотя бы в одном из решений используется класс:
    к общему количеству баллов по итогам хакатона прибавляется +2 Балла

Задание 1 Динамика: Нарисуй квадрат, который меняет свой цвет с некоторой периодичностью. + 2 Балла

Используя графический модуль Turtle в Python, создайте программу, которая рисует на экране квадрат. Данный квадрат раз в пол секунды меняет цвет на другой.

Для паузы между сменой цветов можно воспользоваться функцией sleep из модуля python.

Задание 2 Узор: Создай узор, как в примере. + 2 Балла

Ты можешь использовать любые цвета, но форма рисунка и толщина линий должны быть такими же, как в примере.

pensize = 2

Задание 3 Игра: Создай игру-гонку, которая запускается по нажатию на клавишу пробел. + 2 Балла

Создай три объекта черепашки, по нажатию на пробел они должны начать двигаться с разной скоростью, заметь, что каждый раз скорость черепашек меняется.

В конце напиши цвет той черепашки, что выиграла гонку. От запуска к запуску победитель может меняться.

Задание 4. Движение: Создай круг, который появляется слева сцены и медленно двигается к правому краю сцены. + 3 Балла

Не забудьте очищать экран после каждого шага анимации.

Задание 5. Управление: Создай круг, которым можно управлять с помощью клавиш-стрелок. + 4 Балла

Для привязки обработки клавиш используйте объект класса Screen.
